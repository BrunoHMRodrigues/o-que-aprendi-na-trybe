Unix & Bash

Unix, Kernel e Shell

Unix: É um Sistema Operacional, usado como base para vários sistemas operacionais subsequentes e por isso considerado o pai dos sistemas operacionais. características: portável, multitarefas.

portável: Pode ser adaptado para os mais diversos sistemas computacionais facilmente

Kernel: É o núcleo do sistema operacional. Responsável por fazer a interação entre o Hardware e os Softwares. Ele gerencia os recursos do sistema, é quem faz a detecção de todo o hardware necessário para funcionamento da máquina quando o computador é ligado e é ele quem administra as principais funções da máquina, além do gerenciamento da memória, processos e arquivos do computador.

O Kernel traduz comandos do usuário em instruções para a máquina

Shell: É a interface entre o sistema operacional, usuário e o núcleo do sistema. Interface linha de comando e interface gráfica de usuário. Função: Ler linha de comando, interpretar, executar comando e devolver seu resultado ao usuário.

Exemplos de Shell: SH, BASH, CSH, TCSH, ZSH

Aplicativos: Softwares usados pelo usuário e maioria dos comandos do Unix

Processos: Tudo que é realizado no Unix: tarefas e comandos realizados. Processos possuem identificadores (PID) que carregam diversas informações sobre eles.

Arquivos: São os arquivos, dispositivos e até mesmo os diretórios.

Organizados em hierarquia de raiz, com seu nível mais alto sendo o "/" (diretório raiz). Arquivos tem uma série de informações atreladas a eles:

Arquivos tem uma série de informações atreladas a eles:

Permissões: Tipo de item e nível de permissão para leitura, escrita e execução do item ou diretório.

Links: Quantidade de ligações que o item possui

Proprietário: Dono do arquivo ou diretório, a pessoa que o criou

Grupo: O grupo ao qual o arquivo ou diretório pertence. Serve para dar permissões a outras pessoas

Tamanho: Tamanho em bytes

Data e Hora: Momento da ultima atualização, criação ou modificação

Nome: Nome do arquivo ou diretório

Terminal

Tabela de Comandos Terminal
Comandos Função Propriedades
ls Mostrar arquivos e pastas do diretório a :mostra todos os arquivos do diretório (incluindo os ocultos) l : Lista permissões, número de entidades, dono, grupo, tamanho do arquivo, data e hora da última atualização e nome do arquivo 1 : Apresenta um arquivo por linha
pwd Mostra o caminho para o diretório atual
cd nome_diretorio Usado para entrar em um diretório nome_diretorio : Entra em um diretório digitado que esteja dentro do diretório atual caminho_diretório_desejado : Se move diretamente para o diretório do caminho digitado . : Diretório atual .. : Retorna para o diretório pai do atual / : Retornar para o diretório raiz do sistema ~ : Ir para o diretório Home do usuário logado ../../ : Ir para o diretório dois níceis acima do atual
mkdir nome_diretorio Cria diretórios e subdiretórios
touch nome_arquivo Cria arquivos com o nome e do tipo digitado
cp nome_arquivo nome_arquivo_copia Copia um arquivo para o diretório atual com o nome e tipo do novo arquivo digitado
cat nome_arquivo.tipo_arquivo Mostra conteúdo de um arquivo > : Atribuir inputs do usuário ao conteúdo do arquivo (ctrl + d para sair) Ao usar o > você estará alterando o conteúdo do arquivo, caso já houvesse algo nele o conteúdo seria substituido >> : Edita o conteúdo do arquivo acrescentando novos valores (ctrl + d para sair) > : Pode atribuir inputs do usuário ao conteúdo do arquivo (ctrl + d para sair)>cat arquivo1 arquivo2 > arquivo3 : Irá criar um novo arquivo com o conteúdo dos arquivos chamados antes do >
mv nome_arquivo caminho_destino Copia um arquivo para o diretório atual com o nome e tipo do novo arquivo digitado f : Arquivo é movido independente se estará substituindo um arquivo já existente i : Arquivo é movido e caso exista um arquivo de mesmo nome ele pergunta se deseja continuar n : Nunca substituirá um arquivo existente Caso mais de uma dessas propriedades seja chamada a última é que será usada
rm nome_arquivo Remove um arquivo existente f : Ignora arquivos e argumentos não existentes e remove sem questionar i : Pergunta antes de apagar o arquivo r ou R : Remove diretórios e seus conteúdos Pode ser utilizado com -f para apagar a pasta e conteúdo dentro dela sem fazer perguntas. Ex. rm -rf nome_diretorio d ou dir : Remove diretório vazio (Equivalente a rmdir)
rmdir nome_diretorio Remove um diretório vazio existente f : Ignora arquivos e argumentos não existentes e remove sem questionar i : Pergunta antes de apagar o arquivo r ou R : Remove diretórios e seus conteúdos Pode ser utilizado com -f para apagar a pasta e conteúdo dentro dela sem fazer perguntas d ou dir : Remove diretório vazio (Equivalente a rmdir)
less nome_arquivo Exibe conteúdo de um arquivo em partes criando páginas (barra de espaço - muda de página ; q sai da paginação) Dentro da janela do less /palavra faz uma pesquisa da palavra no conteúdo do arquivo (Ucase sensitive) n para seguir entre o conteúdo encontrado
head nome_arquivo Mostra as 10 primeiras linhas de um arquivo numero_desejado : mostra uma quantidade de linhas iguais ao numero digitado
tail nome_arquivo Mostra as 10 últimas linhas de um arquivo numero_desejado : mostra uma quantidade de linhas iguais ao numero digitado
grep palavra_buscada nome_arquivo Mostra o resultado da busca pela palavra digitada dentro do conteúdo do arquivo (Ucase sensitive) i : Torna a busca independente de maiusculo ou minúsculo Caso queira pesquisar uma frase a escreva entre aspas simples ou duplas v : Trará os valores que não correspondem a pesquisa n : Mostra a linha onde foi encontrada a busca junto com seu conteúdo c : Mostra a quantidade de vezes que o conteúdo foi encontrado no conteúdo do arquivo
wc nome_arquivo Mostra a quantidade de linhas, palavras e caracteres do conteúdo de um arquivo w : Mostra apenas a quantidade de palavras l : Mostra apenas a quantidade de linhas c : mostra apenas a quantidade de caracteres
sort Classifica o conteúdo de um arquivo < arquivo : Mostra o conteúdo do arquivo ordenados sem modificar a ordem dentro do arquivo (forma de output) < arquivo > arquivo_ordenado: Cria um arquivo com o conteúdo ordenado
who Mostra que usuário esta usando o sistema
chmod Altera permissões de acesso de arquivos u : alterações para o usuário g : alterações para o grupo o : alterações para outros usuários a : alterações para todos os usuário + : aumentar permissão - : retirar permissão = : deixar permissões assim como digitadas r, w, x : qual a permissão que você esta alterando Ex: chmod u-rw : retira as permissões r e w do usuário
ps Mostra os processos sendo executados
find . Pesquisa no diretórios por arquivos e pastas de acordo com os parâmetros digitados name "texto":pesquisa pelo texto type d ou type f : pesquisa por todos os diretórios ou arquivos, respectivamente
history Mostra os histórico de comandos utilizados no terminal
echo "texto" Utilizado para exibir mensagens na tela ou em um arquivo > arquivo :Insere o conteúdo digitado em um arquivo

As propriedades podem ser utilizadas colocando um "-" após o comando e mais de uma propriedade pode ser utilizada ao mesmo tempo, independente da ordem de chamada dos mesmos. Ex: ls -1a ou ls -a1

Caso queira mais informação sobre um comando você pode usar: man comando para exibir o manual do comando ou comando --help para ver mais informações sobre ele

whatis comando mostra o que o comando faz; apropos termo busca todos os comando que possam ter relação com o que foi digitado Ex: apropos copy

Coringas: ? pode ser utilizado para substituir um caractere, enquanto * substitui qualquer número de caracteres, sendo muito úteis ao fazer pesquisas pelo terminal. Ex.: ls ?rybe.txt traria os arquivos e pastas do diretório atual que começassem com qualquer caracter desde que após ele houvesse rybe.txt ; ls trybe* traria todos os arquivos que seus nomes começassem com trybe independente do que viesse depois

| pode ser usado para passar o output de um comando para um novo comando Ex: who | sort : resultado do who é enviado para o comando sort que o ordena e o mostrará
